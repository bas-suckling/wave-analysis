# Wave Analysis

### First crack at analysing garmin GPS (.gpx) data for surf sessions.

Session Data:
 - Number of waves
 - Distance surfed
 - Distance Paddled
 - Fastest Speed
 - Total time surfing
 - Average paddle speed
 - Average surf speed

Specific Wave Data:
- Distance surfed
- Max speed

### Data Presentation

Graphs / charts could be made using chartist.js
https://gionkunz.github.io/chartist-js/
